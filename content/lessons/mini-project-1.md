## Tujuan Belajar (Project Lab 1)
Ini adalah proyek pertamamu! Setelah menyelesaikan materi ini, kamu akan bisa **menyusun struktur UI statis menggunakan widget layout yang telah kamu pelajari (Row, Column, Container, dll).**

---

## 1. Instruksi Proyek
Tugasmu adalah membuat **Profile Card Layout** sederhana.
Di lingkungan lokal (komputermu), kamu akan diminta untuk menyusun kode agar aplikasimu menampilkan:
1. Sebuah foto/ikon profil di bagian atas (menggunakan `CircleAvatar` atau `Icon` besar).
2. Sebuah nama dalam format tebal (Bold Text) persis di bawah ikon.
3. Sebuah teks pekerjaan/jabatan di bawah nama.
4. Sebuah deretan ikon kontak di bawahnya secara horizontal (menyamping).

## 2. Apa yang divalidasi?
Sistem (Grader) akan mengecek kodemu secara otomatis. Kamu akan *passed* jika:
- Kamu menggunakan kombinasi `Column` untuk menyusun nama dan jabatan.
- Kamu menggunakan `Row` untuk menyusun ikon-ikon kontak secara menyamping.
- Kodemu bisa berjalan tanpa *error* sintaks.

---

## 3. Langkah Berikutnya
Kamu tidak bisa menulis kodenya di browser ini. 
1. Unduh (download) *starter source code* melalui tombol di sekitarmu.
2. Buka di editor kodemu sendiri (seperti VS Code).
3. Kerjakan instruksinya di dalam file `main.dart`.
4. Jika sudah selesai, *submit* (unggah) kembali ke sini.

*Semangat! Jangan menyerah jika kode pertamamu belum sempurna.*
