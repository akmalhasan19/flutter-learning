# First Flutter App

Sekarang saatnya Anda membuat aplikasi Flutter yang utuh! Di sebelah kanan, Anda dapat melihat instruksi untuk membuat project pertama Anda.

Aplikasi ini akan menggabungkan konsep-konsep dasar yang telah kita pelajari sebelumnya.

## Tujuan Project
1. Membuat `MaterialApp`
2. Menampilkan `Scaffold` dengan `AppBar`
3. Menampilkan teks di tengah layar

Semoga berhasil!
