# Everything is a Widget

Dalam Flutter, hampir semua yang Anda lihat di layar adalah sebuah Widget. 
Mulai dari tombol, teks, hingga layout keseluruhan.

## Widget Tree
Flutter membangun UI dengan menyusun widget di dalam widget lainnya, membentuk sebuah "pohon" (tree) hierarkis.

Mari kita pelajari tipe-tipe widget utama di bagian selanjutnya.
