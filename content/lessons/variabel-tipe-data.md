# Variabel & Tipe Data

Di Dart, semua adalah objek. Saat Anda membuat sebuah variabel, Anda menyimpan referensi ke sebuah objek.

## Deklarasi Variabel

Anda bisa menggunakan `var` agar Dart menentukan tipe data secara otomatis:

```dart
var name = 'Budi';
var year = 2026;
var isLearning = true;
```

Atau mendeklarasikannya secara eksplisit:

```dart
String name = 'Budi';
int year = 2026;
bool isLearning = true;
```

Coba jalankan kode di editor untuk melihat hasilnya!
